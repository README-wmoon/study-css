Transition 속성

어떤 속성을 어떤 방식으로 움직일지 정의할 때 사용하는 속성

transition-duration을 설정하지 않았을 때 기본값 0s

transition-duration:200ms 로 적용했을 때

transition-property

서서히 변화를 주고싶은 css 요소를 설정 예를 들어 배경색상도 변화, 가로길이도 변화, 테두리 둥굴기 변화가 있을 떄 서서히 변화 하는 디자인을 따로 설정하고 싶을 때

transition-property:css속성이름

transition-property 속성을 사용하지 않았을 때는 기본적으로 all이 작용된다. 디자인이 바뀌는 모든 css 가 transition-duration에 걸쳐서 서서히 변화

transition-property:width 변화하는 css 요소 중에서 width 속성만 transition-duration에 걸쳐서 서서히 변화시키겠다(border-radius, background-color는 불연속적으로 뿅 하고 바뀜)

transition-timing-function

서서히 변화시키는 것은 변화시키는 것인데, 속도를 어떻게 설정할지 알려주는 속성 (일정한 속도로 변화, 처음에는 천천히 끝에는 빠르게, 천천히 시작 빠르게 했다가 다시 천천히,...)
